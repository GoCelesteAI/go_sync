# Go Tutorial - Lesson 27: sync Package

This repository contains the source code for **Go Tutorial for Beginners #27 - sync Package**.

## Files

- `sync_mutex.go` - Mutex for exclusive locking and protecting shared data
- `sync_rwmutex.go` - RWMutex for read-heavy concurrent access
- `sync_once.go` - Once for one-time initialization (singleton pattern)
- `sync_pool.go` - Pool for object reuse and reducing allocations

## Running the Examples

```bash
go run sync_mutex.go
go run sync_rwmutex.go
go run sync_once.go
go run sync_pool.go
```
